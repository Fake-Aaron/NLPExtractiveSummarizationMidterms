# Datasets

Proyek ini menggunakan 2 dataset dari Kaggle

## 1. IndoSum — Indonesian Text Summarization
- **Source:** https://www.kaggle.com/datasets/linkgish/indosum
- **Format:** `.jsonl` files split into train/dev/test

## 2. BBC News Summary — English Text Summarization
- **Source:** https://www.kaggle.com/datasets/pariza/bbc-news-summary
- **Format:** `.txt` files organized by category (business, entertainment, politics, sport, tech)

## Kaggle API Setup
Untuk mendownload dataset ke notebook:
1. Masuk ke kaggle.com → Profile → Settings → API → Create New Token (Legacy)
2. Download `kaggle.json`
3. Upload file tersebut saat diminta di Cell 9 notebook
